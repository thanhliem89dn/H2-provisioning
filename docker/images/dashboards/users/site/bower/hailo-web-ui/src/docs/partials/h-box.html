H Box Header

Sass mixin that provides a container with absolutely positioned header and footer.

Don't add any padding and margin to .h-box-header, .h-box-content, .h-box-footer as this will break the layout.

// in your sass file
// params:
//   - $headerHeight - the height of the header on the desktop
//   - $mobileHeaderTop - on the mobile layout header has position:fixed
//                        and needs to be pushed down by this value
//   - $footerHeight - the height of the footer
//   - $mobileHeaderHeight - the height of the header on the mobile

.your-box {
    @include h-box($headerHeight: 40px, $mobileHeaderTop: 40px, $footerHeight: 50px, $mobileHeaderHeight: 50px);
}
