Basics

Use the .h-card to create a basic card. Forgive the verboseness. 

<div class="h-card">
    <div class="hc-title">
        <i class="icon-apple">Holla</i>
    </div>
    <div class="hc-section">
        <div class="hc-c-row">
            <div class="hc-c-c">
                This takes up the whole row
            </div>
        </div>
        <div class="hc-c-row">
            <div class="hc-c-l">
                Left
            </div>
            <div class="hc-c-r">
                Right
            </div>
        </div>
    </div>
    <div class="hc-section">
        <div class="hc-c-row">
            <div class="hc-c-c">
                Worrrrrrrd - another section!
            </div>
        </div>
    </div>
</div>
